# Changelog

All notable changes to this project are documented here. The format follows
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/). This site deploys
continuously, so entries accumulate under [Unreleased].

## [Unreleased]

### Added

- uv build environment: `pyproject.toml` and `uv.lock` pinning zensical
  (`>=0.0.45,<0.1`). The scaffold builds with
  `uv run zensical build --clean --strict`.
- Minimal `zensical.toml` and a placeholder home page.
- Updates section: a static stub page at `docs/updates/index.md`, linked from
  the nav. The dated dev-update log will be generated by a separate
  `zensical-updates` sidecar (a clean-room CC0 rewrite, own repo) once it ships.

### Notes

- The `zensical-blog` plugin spike concluded that a single-build plugin is not
  yet possible (zensical 0.0.45 does not discover `zensical.plugins` entry points
  or call plugin hooks), and the upstream sidecar mutates committed files and
  emits post links that do not match where zensical renders them. We will build
  our own generator instead, so the plugin dependency is dropped for now.
